# 🐾 PawFinds: A Pet Adoption System | MERN Stack Portfolio with Admin Panel

**PawFinds** is a full-stack MERN web application where users can either **put a pet up for adoption** or **adopt a pet**.
If someone can no longer care for their pet, they can submit the pet details through a form. The **admin reviews the request**, and if approved, the pet is listed on the platform for adoption.

For adopters, users can browse pets, apply for adoption, and submit their details. The admin reviews all requests, selects the most suitable adopter, and maintains the adoption history.

This platform makes the **pet adoption journey simple, organized, and trustworthy**.

## 📌 Introduction

PawFinds is designed to connect **pet lovers with pets in need of a loving home**.
The platform provides a smooth workflow for:

* pet submission
* admin approval
* pet listing
* adoption request handling
* adoption history management

## ✨ Features

* 🐶 Users can submit pets for adoption
* 🛡️ Admin approval/rejection workflow
* 📋 Pet listing with profile details and images
* 🔍 Search and filter by pet type (dog, cat, fish, etc.)
* ❤️ Adoption request form for interested adopters
* 👨‍💼 Admin selects the best adopter
* 🗂️ Maintains adoption history records
* 💻 Clean MERN full-stack architecture
* 🔐 Secure backend with MongoDB integration

## ⚙️ Installation Guide

Follow these steps to run the project locally:

### 1) Clone the Repository

```bash
git clone https://github.com/sanket52/final_year_2026.git
```

### 2) Install Dependencies

For both client and server:

```bash
npm install
```

### 3) Configure Environment Variables

Create a `.env` file inside the **server** folder:

```env
mongooseURL=your_mongodb_connection_string
```

### 4) Run Backend

```bash
nodemon server
```

### 5) Run Frontend

```bash
npm start
```
